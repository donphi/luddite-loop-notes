## Feature Overlap Strategy


1. Query UK Biobank vectors for phenotype-related papers
2. Extract phenotype features using NER and pattern matching
3. Create feature embeddings from UK Biobank
4. Search PubMed vectors for similar feature patterns
5. Calculate overlap scores using cosine similarity
6. Validate with Monarch KG relationships
7. Statistical testing of feature significance

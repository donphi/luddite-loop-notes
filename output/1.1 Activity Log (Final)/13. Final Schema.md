Line L19: [body]  ──┐

### Cell Filtering Patterns




~~~python
# Column headings (variable names)
headers = [c for c in table["cells"] if c["is_header"]]

# Row headers (categories)
row_headers = [c for c in table["cells"] if c["is_row_header"]]

# Data cells (actual values)
data = [c for c in table["cells"]
        if not c["is_header"] and not c["is_row_header"]]

~~~

## NER Highlighting Workflow




~~~{=html}
<pre class="notion-ascii-diagram"><code>1. SEARCH/NER finds &quot;hypertension&quot;
   └── block_id: &quot;B17&quot;

2. BLOCK → LINES lookup
   └── blocks[B17].line_token_ids → [&quot;L901&quot;, &quot;L902&quot;]

3. LINE → WORD lookup
   └── lines[L902].words[] → find text == &quot;hypertension&quot;
   └── word.bbox → [245, 892, 312, 918]

4. RENDER
   └── render.pages[page_num].page_file → &quot;page_7.png&quot;
   └── Draw rectangle at [245, 892, 312, 918]
</code></pre>
~~~

~~~{=latex}

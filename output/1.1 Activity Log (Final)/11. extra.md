### 5.2 Recommended Settings for Natural Emergence




~~~{=html}
<pre class="notion-ascii-diagram"><code>Yaml

<strong># RECOMMENDED: Natural emergence of single-disease papers</strong>
disease_ranking:
  max_primary: 3          # Allow natural emergence - don&#39;t force single
  allow_no_primary: true  # Papers with no strong disease focus = excluded

  thresholds:
    primary_min_score: 18.0        # Strict: need title + abstract or witnesses
    secondary_min_score: 6.0       # Moderate bar for secondary
    min_distinct_line_token_ids: 2 # Mentioned on multiple lines
    min_distinct_pages: 1          # At least one page

  zone_weights:
    title: 20.0      # Raise to 20: title mention + witnesses = PRIMARY
    abstract: 5.0    # Keep moderate</code></pre>
~~~

## 8. Configuration Recommendations


### 8.1 For Strict Natural Emergence (Recommended)




~~~{=html}
<pre class="notion-ascii-diagram"><code>disease_ranking:
  max_primary: 3              # Allow 1-3 PRIMARY naturally
  allow_no_primary: true      # OK if no disease qualifies

  thresholds:
    primary_min_score: 18.0   # Need title + abstract or witnesses
    secondary_min_score: 6.0
    min_distinct_line_token_ids: 2
    min_distinct_pages: 1

  zone_weights:
    title: 20.0               # Title with 2 witnesses = 22 &gt; 18 ✓
    abstract: 5.0
</code></pre>
~~~

~~~{=latex}
\begin{Verbatim}[commandchars=\\\{\}]
disease_ranking:
  max_primary: 3              # Allow 1-3 PRIMARY naturally
  allow_no_primary: true      # OK if no disease qualifies

  thresholds:
    primary_min_score: 18.0   # Need title + abstract or witnesses
    secondary_min_score: 6.0
    min_distinct_line_token_ids: 2
    min_distinct_pages: 1

\end{Verbatim}
~~~




This configuration will:


- **Not force** single-disease classification
- Naturally identify ~30-40% of papers as single-primary
- Correctly classify multi-disease papers
- Preserve scientific validity for feature attribution

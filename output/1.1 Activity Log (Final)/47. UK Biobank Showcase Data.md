# UK Biobank Showcase Data


## Content


## 1. Objective and Scope


### 1.1 The Challenge: Building a Ground Truth for Biomedical Feature Mapping


The UK Biobank Showcase data has the schema structure we need to build the ground truth of medical journal research. We must first understand what features the UK Biobank actually measures. The UK Biobank Showcase is the authoritative catalogue of all data fields, encoding systems, and published research that utilises UK Biobank data.







Our objective is to create a comprehensive, machine-readable ground truth that enables:


1. **Feature grounding:** Mapping disease mentions to the specific UK Biobank field IDs they reference
2. **Abbreviation expansion:** Resolving medical acronyms (e.g., "BMI" â†’ "Body Mass Index") for consistent NER
3. **Paper-to-field linkage:** Connecting the frozen corpus of 8,553 publications (July 2025) to the features they analyse

### 3.3 Field Population Coverage and Prediction Modelling Feasibility (EPV updated)


A critical question for large-scale analysis is: how many fields have sufficient participant coverage to be candidates for prediction modelling? The num_participants column in field.txt reveals the actual sample size per field.







However, participant counts alone do not determine prediction-model adequacy. Statistical validity depends on:


- the number of **outcome events** (prevalence / incidence) (no disease / disease),
- the number of **candidate predictor parameters** (not just â€œvariablesâ€),
- the modelling strategy (regression, penalised regression, ML),
- and the internal/external validation design.





Historically, many medical papers referenced Events Per Variable (EPV) rules-of-thumb (e.g., EPVâ‰ˆ10). In current guidance this is treated as too simplistic for modern prediction modelling; the recommended framing is EPP = events per candidate predictor parameter, plus tailored sample size planning for the intended modelling strategy. 



(Riley et al., 2019; Riley et al., 2020; van Smeden et al., 2016)

### 3.4 Implications for Feature Selection


Researchers cannot freely choose from all 10,489 UK Biobank fields. The effective pool of statistically viable features depends on:


1. The outcome being predicted (determines number of events)
2. The **candidate predictor parameter count** (not only â€œnumber of featuresâ€)
3. The modelling method (e.g., penalised regression vs tree ensembles vs deep models)
4. The validation design (resampling, internal-external, external validation)

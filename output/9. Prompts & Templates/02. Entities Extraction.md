# Entities Extraction


## Content


```plain text
Extract biomedical entities from this sentence. Extract ONLY what appears explicitly in the text.


SENTENCE: {raw_sentence}

EXTRACT THESE ENTITY TYPES:

DISEASES: Any medical condition, disorder, syndrome mentioned
- Extract exact disease names as written
- Include abbreviations (T2D, COPD, etc.)

GENES: Gene symbols and names
- Extract exact gene names as written
- Include variants if mentioned (e.g., APOE4)

PROTEINS: Protein names
- Extract exact protein names as written

DRUGS: Medication names, drug classes
- Extract exact drug names as written
- Include brand names and generic names

BIOMARKERS: Clinical markers with their VALUES
- Extract marker name AND its value if present
- Example: "HbA1c of 7.5%" -> {"marker": "HbA1c", "value": "7.5%"}

PATHWAYS: Biological pathways mentioned
- Extract exact pathway names

CELL_TYPES: Specific cell types mentioned
- Extract exact cell type names

ANATOMY: Body parts, organs, tissues
- Extract exact anatomical terms

STATISTICAL_VALUES (extract from paper's results, NOT generate):
- p_values: Extract EXACT p-values from the sentence (e.g., "p=0.003" -> 0.003)
- effect_sizes: Extract effect sizes WITH their type (e.g., "OR=2.3" -> {"type": "OR", "value": 2.3})
- confidence_intervals: Extract CI values (e.g., "95% CI: 1.2-3.4" -> {"level": 95, "lower": 1.2, "upper": 3.4})
- sample_sizes: Extract sample sizes (e.g., "n=1000" -> 1000)
- percentages: Extract percentages mentioned (e.g., "45%" -> 45.0)
- correlations: Extract correlation values (e.g., "r=0.67" -> {"type": "r", "value": 0.67})

UK_BIOBANK_FIELDS: Field identifiers if mentioned
- Format: field_XXXXX or f.XXXXX

COHORTS: Study/cohort names mentioned
- Extract exact cohort names

OUTPUT FORMAT:
{
  "diseases": [],
  "genes": [],
  "proteins": [],
  "drugs": [],
  "biomarkers": [],
  "pathways": [],
  "cell_types": [],
  "anatomy": [],
  "statistical_values": {
    "p_values": [],
    "effect_sizes": [],
    "confidence_intervals": [],
    "sample_sizes": [],
    "percentages": [],
    "correlations": []
  },
  "ukb_fields": [],
  "cohorts": []
}

RULES:
- Extract ONLY what is explicitly written
- Do NOT generate or infer entities
- Return empty arrays for absent types
- For statistical values, extract the EXACT numbers from the paper
```

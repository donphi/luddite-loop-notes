# Stance Extraction


## Content


```plain text
Extract linguistic markers that indicate certainty or uncertainty.


SENTENCE: {raw_sentence}

EXTRACT THESE WORDS/PHRASES:

UNCERTAINTY MARKERS (hedging):
Extract any of these words if present:
- may, might, could, possibly, potentially, perhaps
- probably, likely, unlikely, suggest, indicate
- appear, seem, tend, possible, unclear

CERTAINTY MARKERS:
Extract any of these words if present:
- demonstrate, prove, confirm, establish, show
- definitely, certainly, clearly, obviously
- significant, significantly (in statistical context)

NEGATION:
- Is the word "not" or "no" present?
- What words follow the negation?

CONDITIONAL:
- Is "if" or "when" present?

OUTPUT FORMAT:
{
  "hedging_words": [],
  "certainty_words": [],
  "has_negation": false,
  "negation_context": [],
  "has_conditional": false,
  "conditional_words": []
}

RULES:
- Extract EXACT words from the sentence
- Empty array if no markers found
- Do NOT interpret or score certainty level
```

## Issues & Debugging (15+ Required) — Issue 9: UKB Source Metadata Infected

**summary:** The UK Biobank's metadata has incorrectly listed the publishing year as 2014 for some papers, even those published as recently as 2021, creating a major issue for data reliability. The solution is to rely on the publishing data in the metadata instead of the publishing year.  
**Resolved:** ✗  
**Date:** 2025-08-14  
**Category:** Data Processing  
**Solution:** Major issue, we can not use publishing year confidently and need to instead rely on publishing data in the meta data.
